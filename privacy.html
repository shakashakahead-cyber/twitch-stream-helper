Privacy Policy for Twitch Stream Helper

   

Last Updated: 2025-09-12

   

This browser extension, "Twitch Stream Helper" (hereinafter "the Extension"), is a tool designed to help streamers manage their Twitch stream title, category, tags, and assist with posting to X (formerly Twitter).

   

Information We Collect

   
         
  • Twitch Access Token: We obtain an access token through Twitch's OAuth protocol when you explicitly log in. This token is stored exclusively in your browser's local storage.
    •      
  • Stream Information (Title, Category, Tags): We retrieve and update your own channel information, such as the stream title, category, and tags, using the official Twitch API.
    •    
   

Data Transmission and Sharing

   

The Extension does not transmit your data to any external servers. All communication occurs directly and exclusively with official Twitch domains.

   

Purpose of Use

   
         
  • To retrieve and update your stream title, category, and tags.
    •      
  • To open the X (formerly Twitter) composer window for you to review and post.
    •    
   

Data Storage and Deletion

   

You can delete the access token at any time by using the logout function within the Extension. All locally stored data is permanently deleted when you uninstall the Extension from your browser.

   

Permissions

   

The Extension requires only the `identity` and `storage` permissions. Host permissions are strictly limited to the two necessary Twitch API domains.

   

Third-Party Services

   

This Extension is an unofficial tool and is not affiliated with, endorsed, or sponsored by Twitch Interactive, Inc.

   

Contact Us

   

If you have any questions about this Privacy Policy, please contact the developer at: shakashakahead@gmail.com

 
 
   
   

プライバシーポリシー（Twitch Stream Helper）

   

最終更新日: 2025-09-12

   

本拡張機能「Twitch Stream Helper」（以下「本拡張」）は、配信者が Twitch の配信タイトル・カテゴリ・タグの管理や X（旧Twitter）への投稿を補助するためのツールです。

   

取得する情報

   
         
  • Twitch アクセストークン: ユーザーが明示的にログイン操作を行ったとき、Twitch の OAuth を通じて取得し、ブラウザのローカルストレージにのみ保存します。
    •      
  • 配信情報（タイトル、カテゴリ、タグ）: Twitch API からユーザー自身のチャンネル情報を取得・更新します。
    •    
   

送信・共有

   

本拡張は外部サーバーにデータを送信しません。通信先は Twitch のみです。

   

利用目的

   
         
  • 配信タイトル・カテゴリ・タグの取得/更新
    •      
  • X（旧Twitter）投稿画面の呼び出し（ユーザー確認後の投稿）
    •    
   

保管と削除

   

ログアウト操作でアクセストークンを削除できます。拡張を削除すればローカルのデータも消えます。

   

権限

   

`identity` と `storage` のみを使用。ホスト権限は Twitch の2ドメインに限定。

   

第三者サービス

   

本拡張は Twitch の非公式ツールです。

   

お問い合わせ

   

開発者連絡先: shakashakahead@gmail.com
